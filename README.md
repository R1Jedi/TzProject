User Management System

О проекте
Это приложение для управления пользователями с системой регистрации, авторизации и разграничения прав доступа.

Как это работает
Регистрация и вход
Пользователь регистрируется по username и email

При регистрации автоматически присваивается роль User

Вход в систему осуществляется по username и паролю

Управление профилем
После входа пользователь может:

Редактировать свои данные (имя, фамилию, email)

Менять пароль

Удалить свой аккаунт (мягкое удаление)

Роли пользователей:

User	Управлять своим профилем

Admin	Всё что может User + управление другими пользователями
API для администраторов

Если у вас роль Admin, доступны дополнительные возможности:

Просмотр списка всех пользователей

Назначение других пользователей администраторами

Снятие прав администратора

Что происходит при удалении аккаунта
Аккаунт не удаляется полностью из базы, а просто деактивируется (is_active = False). Пользователь больше не может войти, но данные сохраняются.
